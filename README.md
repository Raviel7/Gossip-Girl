# Gossip Girl ☕

Um projeto divertido e interativo onde você conta uma fofoca e recebe outra — seja de outro usuário ou pescada da internet (Reddit).

## 🚀 Como usar

1. Clone este repositório ou baixe o `.zip`.
2. Abra o arquivo `index.html` em qualquer navegador moderno.
3. Escreva uma fofoca. Clique em "Enviar".
4. Receba uma nova fofoca surpresa.

## 🧠 Tecnologias

- HTML, CSS e JavaScript puro
- Integração com API pública do Reddit
- Armazenamento local com `localStorage`

## 🌐 Hospedagem

Pode ser hospedado facilmente em:

- GitHub Pages
- Netlify
- Vercel
